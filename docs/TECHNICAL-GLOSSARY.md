## How to Use This Glossary

- **KEEP IN ENGLISH** = Do not translate (product names, certifications, acronyms)
- **TRANSLATE** = Use appropriate technical term in target language
- **CONTEXT-DEPENDENT** = Decision based on target market (e.g., "Engineering" may stay in English for some markets)

## 📋 Translation Priority by Language

### Vietnamese (VI) - HIGHEST PRIORITY
**Reason**: Vietnam facility opening April 2026  
**Special Requirements**:
- All diacritics required (á, à, ả, ã, ạ, etc.)
- Formal B2B language (not casual)
- Technical terms often borrow from English/French (acceptable)
- Metric system only (°C, bar, mm)
- Currency: VND (₫)

**Key Vietnamese Technical Terms**:
- Sensor = Cảm biến
- Temperature = Nhiệt độ
- Humidity = Độ ẩm
- Pressure = Áp suất
- Monitoring = Giám sát
- Calibration = Hiệu chuẩn
- Building Automation = Tự động hóa tòa nhà

---

### Japanese (JA) - HIGH PRIORITY
**Reason**: Large HVAC market, technical audience  
**Special Requirements**:
- Formal business language (敬語 keigo)
- Katakana for foreign/technical terms: センサー (sensor)
- Kanji for core concepts: 温度 (temperature), 湿度 (humidity)
- Date format: YYYY年MM月DD日

---

### Chinese (ZH) - HIGH PRIORITY
**Reason**: Manufacturing hub, growing building automation market  
**Special Requirements**:
- Simplified Chinese (mainland China)
- Professional engineering terminology
- Numeric separators: Western format (1,000)
- Currency: CNY (¥)

---

### German (DE), French (FR), Spanish (ES) - MEDIUM PRIORITY
**Reason**: European markets, established HVAC industries  
**Special Requirements**:
- Formal "Sie" / "vous" / "usted" for B2B
- Technical terms well-established in each language
- Regional variants: Swiss German, Quebec French, Latin American Spanish

---

### Arabic (AR) - SPECIAL HANDLING REQUIRED
**Reason**: Middle East markets (UAE, Saudi Arabia)  
**Special Requirements**:
- Modern Standard Arabic (MSA) for business
- **RTL (right-to-left) CSS required**
- Technical terms may use English transliteration
- Currency: AED (د.إ)
- Numbers: Use Western numerals (123) not Eastern (١٢٣)

## 🔍 QA Checklist for Translators

- [ ] Brand names (BAPI, WAM™, Blu-Test) remain in English
- [ ] Certifications (NIST, ISO 9001:2015) remain in English
- [ ] Chemical formulas (CO₂, VOC) remain in English
- [ ] Measurement units (°C, PSI, bar) use symbols only
- [ ] Placeholders preserved: `{year}`, `{count}`, `{count, plural, =1 {...} other {...}}`
- [ ] HTML entities preserved (if any)
- [ ] JSON formatting valid (no syntax errors)
- [ ] Diacritics correct for language (á, ñ, ü, etc.)
- [ ] Formal/professional tone maintained (B2B engineering context)
- [ ] Technical terminology consistent throughout
- [ ] Date/currency formats match regional standards
